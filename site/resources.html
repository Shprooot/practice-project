Используемые инструменты

2ГИС API

Мощный инструмент для работы с картографическими данными, который предоставляет доступ к точным картам, геокодированию и маршрутизации. API позволяет интегрировать картографические сервисы 2ГИС в различные приложения и веб-сайты.

Подробнее об API

Sweet Home 3D

Бесплатное приложение для дизайна интерьера, которое помогает создавать планы помещений, расставлять мебель и просматривать результаты в 3D. Программа предлагает обширную библиотеку 3D моделей и текстур.

Перейти на сайт

Blender

Профессиональный пакет для создания трёхмерной компьютерной графики. Включает в себя средства моделирования, анимации, рендеринга, постобработки видео, а также создания игр. Является свободным и открытым программным обеспечением.

Пожарные планы этажей

Важный источник информации для создания точных планов помещений. Планы эвакуации, размещенные в свободном доступе на этажах университета, предоставляют детальную схему расположения помещений, что значительно упрощает процесс измерений и моделирования.

Наш партнер

2ГИС

2ГИС — это городской информационный сервис, который сочетает в себе бесплатный справочник организаций, точную карту города и навигатор. Сервис предоставляет актуальные данные об организациях, их адресах, контактной информации, отзывах и фотографиях, а также строит маршруты для пешеходов, автомобилистов, велосипедистов и пользователей общественного транспорта.
